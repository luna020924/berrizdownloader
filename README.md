# BerrizDown

BerrizDown은 베리즈 링크를 넣어 영상을 다운로드할 수 있는 안드로이드 앱입니다.

앱을 열고 Berriz에 로그인한 뒤, 영상 링크를 붙여넣으면 썸네일을 먼저 확인하고 다운로드할 수 있습니다. 다운로드한 영상은 기본적으로 사진첩에서 바로 볼 수 있게 저장됩니다.

## 받을 수 있는 파일

지금은 안드로이드 APK만 올려두었습니다.

GitHub의 **Releases** 탭에서 APK를 내려받아 설치하면 됩니다.

- `BerrizDown-0.3.2-android-debug.apk`: 바로 설치해서 테스트하기 좋은 APK
- `BerrizDown-0.3.2-android-release-unsigned.apk`: 직접 서명해서 쓰려는 사람을 위한 APK

일반 사용자는 보통 `debug.apk`를 받으면 됩니다. 안드로이드에서 "알 수 없는 앱 설치" 허용이 필요할 수 있습니다.

## 주요 기능

- Berriz 영상 링크 붙여넣기
- 다운로드 전에 썸네일과 제목 확인
- 영상 다운로드 후 사진첩에 저장
- 썸네일 따로 저장
- 백그라운드 다운로드와 진행률 알림
- 시스템 설정 / 화이트 / 다크 테마 선택

## 사용 방법

1. APK를 설치합니다.
2. 앱 안에서 Berriz에 로그인합니다.
3. 다운로드하고 싶은 Berriz 링크를 붙여넣습니다.
4. 썸네일을 확인한 뒤 다운로드합니다.
5. 완료되면 사진첩에서 파일을 확인합니다.

## 알아둘 점

- 로그인은 앱 안의 WebView를 통해 처리됩니다.
- DRM이 걸려 있거나 권한이 없는 영상은 받을 수 없을 수 있습니다.
- 네트워크 상태와 영상 길이에 따라 다운로드 시간이 달라질 수 있습니다.
- 이 앱은 개인적인 백업이나 본인이 이용할 권한이 있는 콘텐츠에만 사용해야 합니다.

## 직접 빌드하기

Android Studio에서 이 폴더를 열거나, 터미널에서 아래 명령을 실행하면 됩니다.

```bash
./gradlew :app:assembleDebug
```

빌드된 APK는 `app/build/outputs/apk/debug/` 안에 만들어집니다.
